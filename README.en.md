# SpringBoot-Vue-032_蛋糕

#### Description
基于SpringBoot + MyBatis Plus + Vue.js的B/S架构电商管理系统，实现蛋糕商品展示、在线购物车、订单管理、用户权限控制等核心功能。系统采用前后端分离设计，后端提供RESTful API接口，前端使用ElementUI组件库构建用户界面。
